Generative AI Query Engine

An AI-powered query engine built with LangChain, Hugging Face LLMs, Wikipedia API, and SQL to handle both structured and unstructured data.
It allows users to ask natural language questions and get direct answers from databases or Wikipedia knowledge.

Features

LLM-Powered Querying – Uses Hugging Face LLMs with LangChain for reasoning

Wikipedia Integration – Real-time knowledge extraction and summarization

SQL Query Generation – Converts natural language into SQL queries for structured data

Unstructured Data Handling – Summarizes and answers from raw text

Tech Stack

Python 3.10+

LangChain

Hugging Face Transformers

Wikipedia API

SQL (SQLite / PostgreSQL)
